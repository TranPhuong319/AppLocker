# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly by following the steps below:

1. **Report Location**  
   Submit details about the vulnerability to our security team [through GitHub](https://github.com/TranPhuong319/AppLocker/security/advisories/new)  

2. **Information to Provide**  
   Include the following information to help us understand and address the issue quickly:  
   - Detailed description of the vulnerability  
   - Steps to reproduce the issue  
   - Impact assessment (if known)  
   - Any suggested fixes or patches (optional)  

3. **Response Timeline**  
   - You can expect an initial acknowledgment of your report within 48 hours.  
   - We will provide updates on progress and expected resolution timelines as soon as possible.  

4. **Disclosure Policy**  
   - We commit to keeping reports confidential until a fix is available.  
   - Coordinated disclosure will be done after patches are released or mitigations are in place.

5. **Thank You**  
   We appreciate your effort in helping us improve the security of this project.

---

*This Security Policy will be updated periodically to reflect any changes in supported versions or reporting procedures.*
