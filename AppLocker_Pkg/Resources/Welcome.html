Welcome to the AppLocker Installer

AppLocker helps you lock applications on macOS securely and conveniently. This installer allows you to choose between Install and Uninstall.

Important Notice for Uninstall

Before uninstalling AppLocker, please make sure to unlock all applications that were previously locked. If you do not unlock them first, you may encounter issues opening these applications after AppLocker has been removed.

The uninstall process will automatically stop background services and remove all AppLocker components from your system.
